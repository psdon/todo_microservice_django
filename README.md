# app

Todo Microservice

# Prerequisites

- [Docker](https://docs.docker.com/docker-for-mac/install/)  

# Local Development

Start the dev server for local development:
```bash
docker-compose build
```

```bash
docker-compose up
```

Run a command inside the docker container:

```bash
docker-compose run --rm todo [command]
```
